## Derived Agent Rules

- `.pyi` stubs are parse-clean and are the densest source-adjacent callable reference layer found under `/Parashell`.
- Not every `.py` file under `/Parashell` parses under the active interpreter. Do not bulk import arbitrary source files as a discovery tactic.
- Runtime discovery should prefer loaded modules, `.pyi` signatures, object properties, and document-supported TypeIds over direct source-file imports.
- GUI-facing source repeatedly gates imports and viewprovider behavior behind `FreeCAD.GuiUp`; headless snippets must preserve that gate.
- Transaction behavior is document-visible but can be application-scoped through `setActiveTransaction`; multi-document edits require extra care.
- Tests under `/Parashell/src/Mod` are useful operational evidence for recompute, alias, expression, shape, TechDraw, and GUI-boundary behavior.
